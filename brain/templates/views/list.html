Tranco+Passkey List

About this list

We provide a continuously updated list of authentication mechanisms found on millions of websites in the Internet. When configuring this list with a specific date, we retrieve the Tranco list that contains the 1 million most popular domains for that date. For each domain in the Tranco list, we supplement our latest data up to this date.

Update Cycle

While Tranco provides daily lists, we need about a month to snapshot the authentications on all of the 1 million websites. In the worst case, we can only provide data to an entirely new domain that has been added to the Tranco list one month later. However, most of the Tranco domains remain stable and only move up or down in their ranking.

What data does this list include?

The data in our list provides comprehensive details about passkey adoption and authentication mechanisms found on websites. For each domain, the list includes:

  • Login Page URLs: Detected login page URLs with their discovery strategies (PATHS, HOMEPAGE, CRAWLING, etc.)
  • Passkey Detection: Whether passkeys are detected, detection methods used (UI, JavaScript API, Enterprise patterns, Keywords, Metadata), confidence levels, and detection indicators
  • WebAuthn Implementation Parameters: Detailed WebAuthn API calls including create/get options, credentials, CDP events, and implementation details captured via virtual authenticator
  • Element Information: Passkey button coordinates, dimensions, inner text, outer HTML, element tree, and validity status
  • Authentication Mechanisms: Support for Passkey, Multi-Factor Authentication (MFA), and Password-based authentication
  • Identity Providers: Detected SSO providers (Google, Microsoft, Apple, GitHub, Facebook) if present
  • Artifacts: References to screenshots, HAR files, and other artifacts stored in object storage

The data structure includes both landscape analysis (passkey presence detection) and passkey analysis (detailed WebAuthn implementation parameters) results.
